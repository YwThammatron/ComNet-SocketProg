# Computer Network Socket Programming Assignment

Tools and Languages
- Python

Prerequisites
- Binary file

**Assignment Goals**
- Program has to reliably send file from client process to server process due to packet duplication or packet loss
- Program is able to send binary file
- Program has to accept indicating IP address and server port via command line
- Program is able to operate under Linux OS and be developed by python 3.8+
- Installing other software such as libraries or modules is prohibited
- Program has to use only UDP protocol in Transport layer
- Receiving file in server side use only one socket
